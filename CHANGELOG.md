# Changelog

## v0.1.0 - Initial Beta

- Added CEL-to-entgo SQL predicate compilation for a declared schema.
- Supported comparisons, boolean logic, membership lists, string helpers, null checks, converters, custom predicate builders, and relation `exists` filters.
- Added structured errors for unknown fields, unsupported expressions, invalid literals, relation filter problems, and converter failures.

This is a beta API intended for early integration and feedback.
